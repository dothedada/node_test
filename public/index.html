Esta es una prueba de servidor en Node para sitios estáticos

¿Qué es un servidor en Node.js para sitios estáticos?

Un servidor en Node.js para sitios estáticos es una aplicación ligera que entrega archivos HTML, CSS, JavaScript e imágenes directamente al navegador sin procesamiento adicional. Esto lo convierte en una opción eficiente para servir contenido rápido y seguro.

¿Por qué usar Node.js para esto?

Node.js es rápido, escalable y fácil de configurar. Con módulos como http o express, puedes crear un servidor en pocos minutos y manejar peticiones con bajo consumo de recursos. 

const express = require("express");
const app = express();

app.use(express.static("public"));
app.listen(3000, () => console.log("Servidor en http://localhost:3000"));
